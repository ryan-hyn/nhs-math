1. Quadratic Graphs and Their Properties

A quadratic function is a function where the highest exponent is 2.

The standard form of a quadratic function is:

y = ax² + bx + c

or

f(x) = ax² + bx + c

Important Properties

Property Meaning
Parabola The U-shaped graph of a quadratic function
Vertex The highest or lowest point of the parabola
Axis of Symmetry The vertical line that cuts the parabola in half
Opens Upward Happens when a is positive
Opens Downward Happens when a is negative
X-Intercepts Where the graph touches or crosses the x-axis
Roots / Zeros Another name for the x-intercepts
Y-Intercept Where the graph crosses the y-axis
If a is positive, the graph opens upward like a smile.
If a is negative, the graph opens downward like a frown.

Example:

y = x² + 5x + 6

This is quadratic because the highest exponent is 2.

2. Solving Quadratic Equations

A quadratic equation is usually written as:

ax² + bx + c = 0

The goal is to find the values of x that make the equation equal 0.

These values are called:

  • zeros
  • roots
  • solutions
  • x-intercepts
Number of Zeros Meaning
2 zeros The graph crosses the x-axis twice
1 zero The graph touches the x-axis once
0 real zeros The graph does not touch the x-axis

3. The Quadratic Formula

The quadratic formula is:

x = (-b ± sqrt(b² - 4ac)) / 2a

You use this formula when solving:

ax² + bx + c = 0

The values are:

a = coefficient of x²

b = coefficient of x

c = constant

4. Example With Two Zeros

Example:

x² + 5x + 6 = 0

Here:

a = 1, b = 5, c = 6

Step 1: Plug the values into the quadratic formula.

x = (-5 ± sqrt(5² - 4(1)(6))) / 2(1)

Step 2: Simplify inside the square root.

x = (-5 ± sqrt(25 - 24)) / 2

x = (-5 ± sqrt(1)) / 2

x = (-5 ± 1) / 2

Step 3: Solve both equations.

First solution:

x = (-5 + 1) / 2

x = -4 / 2

x = -2

Second solution:

x = (-5 - 1) / 2

x = -6 / 2

x = -3

Final Answer: x = -2 and x = -3

This means the graph crosses the x-axis at x = -2 and x = -3.

5. Example With No Real Zeros

Example:

x² + 4x + 8 = 0

Here:

a = 1, b = 4, c = 8

Step 1: Plug the values into the quadratic formula.

x = (-4 ± sqrt(4² - 4(1)(8))) / 2(1)

Step 2: Simplify inside the square root.

x = (-4 ± sqrt(16 - 32)) / 2

x = (-4 ± sqrt(-16)) / 2

Since sqrt(-16) is not a real number, there are no real zeros.

Final Answer: No real solutions

This means the graph does not cross the x-axis.

6. Example With One Zero

Example:

x² - 6x + 9 = 0

Here:

a = 1, b = -6, c = 9

Step 1: Plug the values into the quadratic formula.

x = (-(-6) ± sqrt((-6)² - 4(1)(9))) / 2(1)

Step 2: Simplify.

x = (6 ± sqrt(36 - 36)) / 2

x = (6 ± sqrt(0)) / 2

x = (6 ± 0) / 2

Step 3: Solve both.

x = (6 + 0) / 2 = 3

x = (6 - 0) / 2 = 3

Both answers are the same, so there is only one zero.

Final Answer: x = 3

This means the graph touches the x-axis once at x = 3.

7. Quadratic Functions

A quadratic function can be graphed using its important points.

The standard form is:

y = ax² + bx + c

Finding the Vertex

The vertex is the highest or lowest point of the parabola.

Vertex Formula: x = -b / 2a

Then plug that x-value back into the equation to find the y-value.

The vertex is written as:

(x, y)

Graphing a Quadratic Function

  1. Find the vertex using x = -b / 2a.
  2. Plug that x-value into the equation to find the y-value.
  3. Find the zeros by factoring or using the quadratic formula.
  4. Plot the vertex and zeros on the graph.
  5. Draw a smooth parabola through the points.

8. Factoring to Solve Quadratic Equations

Sometimes quadratic equations can be solved by factoring.

Example:

x² + 5x + 6 = 0

Step 1: Factor the quadratic.

Find two numbers that multiply to 6 and add to 5.

Those numbers are 2 and 3.

So:

x² + 5x + 6 = 0

becomes:

(x + 2)(x + 3) = 0

Step 2: Set each factor equal to zero.

x + 2 = 0

x = -2

And:

x + 3 = 0

x = -3

Final Answer: x = -2 and x = -3

This means the graph crosses the x-axis at x = -2 and x = -3.

Not every quadratic equation is factorable. If it is not factorable, use the quadratic formula.

9. Systems of Linear and Quadratic Equations

A system of linear and quadratic equations has:

  • one linear equation
  • one quadratic equation

A linear equation makes a straight line.

A quadratic equation makes a parabola.

The solutions are the points where the line and parabola intersect.

The main method is substitution because both equations are usually equal to y.

Example:

y = x + 2

y = x² - 2x - 2

Step 1: Set the equations equal to each other.

x + 2 = x² - 2x - 2

Step 2: Move everything to one side.

0 = x² - 3x - 4

Step 3: Factor the quadratic.

Find two numbers that multiply to -4 and add to -3.

Those numbers are -4 and 1.

Split the middle term:

x² - 4x + x - 4

Group the terms:

(x² - 4x) + (x - 4)

Factor each group:

x(x - 4) + 1(x - 4)

Factor out the matching parentheses:

(x + 1)(x - 4)

Step 4: Set each factor equal to zero.

x + 1 = 0

x = -1

And:

x - 4 = 0

x = 4

Step 5: Plug each x-value into the linear equation.

Use:

y = x + 2

For x = 4:

y = 4 + 2

y = 6

So one solution is:

(4, 6)

For x = -1:

y = -1 + 2

y = 1

So the other solution is:

(-1, 1)

Final Answer: (4, 6) and (-1, 1)

These are the two points where the line and parabola intersect.

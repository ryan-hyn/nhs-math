Rate of Change and Slope

Slope Formula

m = (y₂ - y₁) / (x₂ - x₁)

Slope tells us how steep a line is.

slope = rise / run
  • Rise = how much the line moves up or down
  • Run = how much the line moves left or right

Rate of Change

Rate of change tells us how much one quantity changes compared to another.

Rate of Change = change in y / change in x

In Algebra 1, rate of change and slope usually mean the same thing.

Finding Slope

From a Graph

  1. Start at one clear point on the line.
  2. Move to another clear point.
  3. Count how far you move up or down.
  4. Count how far you move right.
  5. Write it as rise over run.

Graphs should be read from left to right.

  • If the line goes up, the slope is positive.
  • If the line goes down, the slope is negative.
  • If the line is flat, the slope is zero.

From a Table

To find slope from a table, divide the change in y by the change in x.

x y
1 3
2 5
3 7
The y-values go up by 2.
The x-values go up by 1.
So the slope is 2 / 1 = 2.

From an Equation

If the equation is in slope-intercept form:

y = mx + b

Then m is the slope.

From Two Points

m = (y₂ - y₁) / (x₂ - x₁)
Points: (2, 4) and (6, 12)
m = (12 - 4) / (6 - 2)
m = 8 / 4
m = 2

Rate of Change Word Problem Example

A car travels 120 miles in 2 hours. What is the rate of change?

rate = distance / time
rate = 120 / 2
rate = 60

The car travels 60 miles per hour.

Slope-Intercept Form

y = mx + b
  • m = slope
  • b = y-intercept
  • y-intercept = where the line crosses the y-axis

Writing an Equation from a Graph

  1. Find the y-intercept.
  2. Find the slope.
  3. Plug them into y = mx + b.
If the slope is 2/3 and the y-intercept is 4:
y = 2/3x + 4

Writing an Equation from a Table

  1. Find the y-intercept, usually when x = 0.
  2. Find the slope using change in y over change in x.
  3. Plug the slope and y-intercept into y = mx + b.

Horizontal and Vertical Lines

A horizontal line looks like:

y = number

A vertical line looks like:

x = number

Standard Form

Ax + By = C
  • A = number in front of x
  • B = number in front of y
  • C = number by itself
Example: 2x + 3y = 12

A = 2
B = 3
C = 12

Writing Standard Form from a Graph

Suppose the graph has a y-intercept of 4 and a slope of 2/3.

First write slope-intercept form:
y = 2/3x + 4

Multiply everything by 3:
3y = 2x + 12

Move the x-term to the left side:
-2x + 3y = 12

Rewriting Standard Form into Slope-Intercept Form

Example: 6y - 3x = 12

Add 3x to both sides:
6y = 3x + 12

Divide everything by 6:
y = 1/2x + 2

Parallel and Perpendicular Lines

Parallel Lines

Parallel lines have the same slope but different y-intercepts.

y = 2x + 5 is parallel to y = 2x + 9

Perpendicular Lines

Perpendicular lines have slopes that are opposite reciprocals.

To find the opposite reciprocal:

  1. Flip the fraction.
  2. Change the sign.
4/7 becomes -7/4

y = 4/7x + 5 is perpendicular to y = -7/4x + 5

Arithmetic Sequences

An arithmetic sequence adds or subtracts the same number each time.

That number is called the common difference.

3, 7, 11, 15, 19
This sequence adds 4 each time.
30, 25, 20, 15, 10
This sequence subtracts 5 each time.

Arithmetic Sequence Formula

aₙ = a₁ + d(n - 1)
  • aₙ = the term you want to find
  • a₁ = the first term
  • d = common difference
  • n = term number
Find the 4th term of 30, 25, 20, 15, 10.

a₁ = 30
d = -5
n = 4

a₄ = 30 + (-5)(4 - 1)
a₄ = 30 + (-5)(3)
a₄ = 30 - 15
a₄ = 15

Geometric Sequences

A geometric sequence multiplies or divides by the same number each time.

That number is called the common ratio.

2, 6, 18, 54, 162
This sequence multiplies by 3 each time.
243, 81, 27, 9, 3
This sequence multiplies by 1/3 each time.

Geometric Sequence Formula

aₙ = a₁(rⁿ⁻¹)
  • aₙ = the term you want to find
  • a₁ = first term
  • r = common ratio
  • n = term number
Find the 3rd term of 2, 6, 18, 54, 162.

a₁ = 2
r = 3
n = 3

a₃ = 2(3³⁻¹)
a₃ = 2(3²)
a₃ = 2(9)
a₃ = 18

Finding the Common Ratio

r = any term / term before it
2, 6, 18, 54

r = 6 / 2
r = 3
243, 81, 27, 9, 3

r = 81 / 243
r = 1/3

Scatter Plots

A scatter plot is a graph with points that show the relationship between two things.

Examples:

  • Hours studied and test scores
  • Height and shoe size
  • Temperature and ice cream sales

Each point represents one piece of data.

Best Fit Line

A best fit line is a straight line that goes through the middle of the data points as closely as possible.

It shows the overall trend of the data.

Positive Relationship

If the line goes upward from left to right, the relationship is positive.

Example: More hours studied usually means higher test scores.

Negative Relationship

If the line goes downward from left to right, the relationship is negative.

Example: More time spent driving may mean less gas left in the tank.

No Relationship

If the points are scattered randomly, there is little or no relationship.

Simple Review

  • Slope tells how steep a line is.
  • Rate of change tells how much y changes compared to x.
  • Slope-intercept form is y = mx + b.
  • Standard form is Ax + By = C.
  • Parallel lines have the same slope.
  • Perpendicular lines have opposite reciprocal slopes.
  • Arithmetic sequences add or subtract the same number.
  • Geometric sequences multiply or divide by the same number.
  • Scatter plots show data points.
  • Best fit lines show the trend of the data.

Unit 2: Solving Equations

Identifying a Solution

To determine whether a value is a solution, substitute it into the equation and check if both sides are equal.

Example:
5x = 40

If x = 8:
5(8) = 40 ✔
Example:
27 + 4x = 51

If x = 6:
27 + 24 = 51 ✔

One-Step Equations

One-step equations require one inverse operation to isolate the variable.

5x = 40
x = 8

Two-Step Equations

27 + 4x = 51

4x = 24
x = 6

Multi-Step Equations

x/2 - 40 = 10 × 3

x/2 - 40 = 30
x/2 = 70
x = 140
Simplify both sides first whenever possible.

Equations with Variables on Both Sides

4x - 16 = 2x

-16 = -2x
x = 8
6x - 10 = 3x + 5

3x - 10 = 5
3x = 15
x = 5
The goal is to move all variables to one side and constants to the other.

No Solution Cases

8y(10 - 6) = 2y(8 + 10)

32y = 36y

If y is only one number, you can't multiple y by two different numbers and get the same results. Therefore, the solution doesn't exist.

Literal Equations and Formulas

A literal equation contains two or more variables. You solve for one variable in terms of the others.

3x + 5y = 20

3x = 20 - 5y
x = (20 - 5y) / 3
ab + c = 2c - 4b

ab = c - 4b
a = (c - 4b) / b
A = bh

b = A / h

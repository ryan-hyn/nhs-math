Identifying a Solution

To determine whether a value fits the solution, substitute it into the equation and check if both sides are equal.


Example:
5x = 40

If x = 8:
5(8) = 40 ✔

Example:
27 + 4x = 51

If x = 6:
27 + 24 = 51 ✔

One-Step Equations

One-step equations require one inverse operation to isolate the variable.
Since we aren't solving the equation, but instead working backwards to find a missing piece of the equation,
we have to use PEMDAS in reverse.


Expression How to isolate the variable
Multiplication Divide the term
Division Multiply the term
Addition Subtract the term
Subtraction Add the term

Note: whatever you do to one side of the equal sign, you have to do to the other side.

Example:
5x = 40 → since the problem multiplies, you divide by 5 on both sides.
x = 8

Example:
x - 8 = 10 → since the problem subtracts, you add 8 to both sides.
x = 18

Two-Step Equations

It's the same process, but remember to use PEMDAS in reverse.

Example:
27 + 4x = 51 → working in reverse, we'll begin by subtracting both sides by 27
4x = 24 → next, we'll divide both sides by 4.
x = 6

Example:
3/4(x) - 0.25 = 3.75 → we'll begin by bringing -0.25 over by adding 0.25 to both sides
3/4(x) = 4 → next, divide both sides by 3/4 in order to isolate x. Remember when dividing a number by a fraction to use Keep, Change, Flip
x = 16/3

Multi-Step Equations

Somtimes, you can simplify one or both sides of the equation before moving terms
to the other side of the equal sign. This will make the process of isolating the variable easier.


Example:
x/2 - 40 = 10 × 3 → since 10 × 3 can already be solved, we should simplify this first
x/2 - 40 = 30 → now we can proceed by subtracting both sides by 40
x/2 = 70 → finally, we can isolate x by multplying both sides by 2
x = 140

Example:
5(2x - 25) = 7x - 9x - 5 → begin by simplifying both sides in order to make it easier later
10x - 125 = -2x - 5 → now, we should separate our variables and constants
12x = 120 → finally, we can isolate x by dividing both sides by 12
x = 10

Example:
5(3x - 1) / 15 = 2/3 - 10x → using the distributive method, lets distribute 5 into 3x and -1
(15x - 5) / 15 = 2/3 - 10x → now, we should divide 15x and 5 by 15, then simplify our answer
x - 1/3 = 2/3 - 10x → now, we should separate our variables and constants
11x = 1 → finally, we can divide both sides 11, isolating x
x = 1/11

Equations with Variables on Both Sides

The goal is to move all variables to one side and constants (numbers) to the other.


Example:
4x - 16 = 2x → subtract 4x from both sides in order to separate 16 and the variable
-16 = -2x → divide both sides by -2
x = 8

Example:
6x - 10 = 3x + 5 → I can move any term over, but we'll begin by moving 3x over by subtracting 3x
3x - 10 = 5 → now, I'll move 10 over by adding 10 to both sides
3x = 15 → divide both sides by 3
x = 5

No Solution Cases

At times, you'll run into problems in which the equation doesn't have a solution.


8y(10 - 6) = 2y(8 + 10) → we'll being by solving the expression in the parentheses
8y(4) = 2y(18) → now, we'll simplify further by multiplying the terms on both sides
32y = 36y

It's impossible to multiply one number by two different numbers and get the same answer.
Therefore, the equation has no solution; y can't be two different numbers.

Literal Equations and Formulas

A literal equation contains two or more variables. You solve for one variable in terms of the others.


Example:
3x + 5y = 20 → let's solve for x. We'll begin by bringing 5y over to the other side by subtracting 5y
3x = 20 - 5y → next, we'll divide both sides by 3, isolating x by itself.
x = (20 - 5y) / 3

Example:
ab + c = 2c - 4b → let's solve for a. We'll being by bringing c over to the other side by subtracting c
ab = c - 4b → next, we'll divide both sides by b, isolating x by itself
a = (c - 4b) / b

Example:
A = bh → this is area = base × height. Let's solve for b by dividing both sides by h
b = A / h

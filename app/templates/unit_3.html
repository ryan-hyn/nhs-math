Unit 3: Introduction to Functions

Relations and Functions

A relation is a pairing of inputs (x-values) and outputs (y-values). A function is a special relation where each input has exactly one output.

Key Terms

  • Domain: all x-values (inputs)
  • Range: all y-values (outputs)
Vertical Line Test: If a vertical line touches a graph more than once, it is not a function.

x-value = input, y-value = output

Using Graphs to Relate Quantities

Graphs show relationships between two variables, often using tables or real-world data.

y = mx + b
  • y = output
  • m = slope (rate of change)
  • x = input
  • b = y-intercept
m = rise / run

Patterns and Linear Functions

A linear function forms a straight line with a constant rate of change.

  • Linear: straight line, constant change
  • Non-linear: curved or changing rate

Graphing a Function Rule

Steps to graph a function:

  1. Create a table of values
  2. Plot ordered pairs (x, y)
  3. Connect points (straight line if linear)
Example:
y = 2x + 1

Substitute values for x, solve for y, and plot points.

Writing a Function Rule

A function rule describes the relationship between x and y. You can also use f(x) instead of y.

f(x) = 3x + 2
Each input (x) produces exactly one output (y).

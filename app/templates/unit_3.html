Relations and Functions

Key Terms

    Function: a special relation where each input has exactly one output
    Relation: a relation is a pairing of inputs (x-values) and outputs (y-values)
    Domain: all x-values (inputs) that go left to right on a graph
    Range: all y-values (outputs) that go up and down on a graph
    x-axis: horizontal line placed at y = 0. It separates the negative y values from the positive
    y-axis: vertical line placed at x = 0. It separates the negative x values from the positive
    x-intercept: the x value when y = 0. It crosses the graph from negative to positive y values
    y-intercept: the y value when x = 0. It crosses the graph from negative to positive x values

This graphed function is represented by, x = y. Looking at the graph, we can tell this by picking an x value and
matching it to the y value that intersects with the graph. On the graph, when x = 2, y = 2. When x = 0, y = 0.


Vertical Line Test

A function doesn't exist if an x value corresponds with two y values. We can use to Vertical Line Test in order to check if a line on a graph is a function or not. The test is to draw a straight vertical line on the graph: if the equation touches the line more than once, then the equation is not a function.


Example: y2 + x2 = 9 (red line)

Drawing a line at x = 1 (blue line), it crosses the graphed equation twice meaning this is NOT a function.


Example:
y = 3x + 6 (blue line)

Drawing a line at x = -1 (green line), it crosses the graphed equation once meaning this IS a function.


Example:
y = √|x| (blue line)

Drawing a line at x = 4 (green line), it crosses the graphed equation once meaning this IS a function.


Using Graphs to Relate Quantities

Graphs aren't the only way to show a relation between x and y values--a table could be used aswell!

Tables

Example: y = -1/2(x) + 3

x -4 -2 0 2
y 5 4 3 2

Looking at the table, you can input any x value you desire into the equation,
and your answer (y output) goes into the other row of the table.

Function vs. Non-function (Table)

Remember, one x value cannot correspond with two y values, but the opposite is ok.


NOT a function
x 13 3 4 4
y 1 8 3 -3

One x value (4) correspond with two y values (3 and -3). This table doesn't represent a function.


IS a function
x -6 -1 5 8
y 18 -2 7 7

Two separate x-values (5 and 8) correspond with 7. This table represents a function.

Match a Table and a Graph

x 0 1 2 3
y 2 5 8 11

How to match a graph and a table


Method 1

Step 1: Calculate how much y increases or decreases when x goes up
- based on the table, every time x goes up by 1, y goes up by 3

Step 2: Match the values of the y-intercept meaning, the y value when x = 0
- based on the table, when x = 0, y = 2. The right graph will show the function crossing the y-axis at y = 2

Combining our findings together, the table should match the second graph because on the graph,
The function crosses the y-axis at y = 2, and every time x goes up by 1, y goes up by 3.


Method 2

Step 1: On each graph, pick an x value, draw a line until it hits the function and see what the y value is
Step 2: Compare your answers from each graph to the table. Whichever one fits is your answer!

Patterns and Linear Functions

Key Terms

    Linear Function: a function that creates a straight line (constant rate of change)
    Non-linear Function: a curved function (varying rate of change)
    Rate of Change: how much y changes compared to how much x changes
    Slope: the rate of change represented by:
    Change in y Change in x

A linear function is represented by the formula:
y = mx + b

    y: y-value
    m: slope
    x: x-value
    b: y-intercept

Linear Function

Example: (look below)
x 0 2 4 6
y 6 14 22 30

Starting from 6, the y value goes up by 8 everytime the x value goes up by 2. This table represents a linear function.

The graph shows a straight line, or a constant rate of change
proving that it represents a linear function

Non-linear Function

Example: (look below)
x 3 4 5 6
y 9 16 25 36

Starting from 9, the y value goes up by various increasing amounts while the x value goes up by 1 each time. This table represents a non-linear function.

The graph shows a curved line, or a various rate of change
proving that it represents a non-linear function


Graphing a Function Rule

Key Terms

    Ordered Pair: two numbers usually (x, y) that mark a coordinate or a point on a graph

From a Table

Remember: one x value cannot have two corresponding y values

Step 1: Create a table of values. One row represents x values, the other row y values
Step 2: Plot ordered pairs (x, y) on a graph by marking the coordinate with a point
Step 3: Connect points with a line (straight line if the function's linear)

From an Equation

Example: y = -2x + 3

Step 1: Select a few x values and plug them into the equation (Recommend using 4 values that are near each other)
Step 2: Using the answers, create ordered pairs and plot the coordinates on the graph
Step 3: Connect points with a line (straight line if the function's linear)

To show an example, I'll use the x values: 1, 2, 3, and 4 and plug it into the last example's equation (y = -2x + 3)

(1, y): -2(1) + 3 = 1
(2, y): -2(2) + 3 = -1
(3, y): -2(3) + 3 = -3
(4, y): -2(4) + 3 = -5
Step 2

Ordered pair at: (1, 1) (2, -1) (3, -3) (4, -5)

Writing a Function Rule

A function is written as either equal to y, or by the function notation: f(x).

f(x) = 3x + 2

Note: since each input (x) produces exactly one output (y), y should never be raised to a power above 1. Doing so means that an x value can have multiple corresponding y values.

Quadratic Graphs and Their Properties

A quadratic function is a function where the highest exponent is 2.

Formula: y = ax² + bx + c

Exponential / quadratic funtion

Important Properties / Terms

Property / Terms Meaning
Parabola The U-shaped graph of a quadratic function
Vertex The highest or lowest point of the parabola
Axis of Symmetry The vertical line that cuts the parabola in half
Opens Upward Happens when a in ax² is positive
Opens Downward Happens when a in ax² is negative
X-Intercepts Where the graph touches or crosses the x-axis
Roots / Zeros Another name for the x-intercepts
Y-Intercept Where the graph crosses the y-axis
Note: an easy way to tell when a is negative or positive
  • When a is positive, the graph looks like a smiley face
  • When a is negative, the graph looks like a frowny face

Solving Quadratic Equations

In order to solve quadratic equations, make the equation equal to 0 → ax² + bx + c = 0

The goal is to find the values of x that make the equation equal 0.

These values are called:

  • zeros
  • roots
  • solutions
  • x-intercepts
Number of Zeros Meaning
2 zeros The graph crosses the x-axis twice
1 zero The graph touches the x-axis once
0 real zeros The graph does not touch the x-axis
Note: the highest exponent in a function is the amount of possible zeros in the equation. However, that doesn't mean there will always be 2 zeros when the highest exponent is 2 for example.

The Quadratic Formula

Formula: x =
-b ± √2b² - 4ac
2a

Example With Two Zeros

Example: x² + 5x + 6 = 0

Step 1: Plug the values into the quadratic formula.

x =
-5 ± √5² - 4(1)(6)
2(1)

Step 2: Simplify

x =
-5 ± √(25 - 24)
2

x =
-5 ± √1
2

x =
-5 ± 1
2

Step 3: Solve both equations.

First solution:

x = (-5 + 1) / 2

x = -4 / 2

x = -2

Second solution:

x = (-5 - 1) / 2

x = -6 / 2

x = -3

Final Answer: x = -2 and x = -3

This means the graph crosses the x-axis at x = -2 and x = -3.

Example With No Real Zeros

Example: x² + 4x + 8 = 0

Step 1: Plug the values into the quadratic formula

x =
-4 ± √4² - 4(1)(8)
2(1)

Step 2: Simplify

x =
-4 ± √-16
2

Since √(-16) is not a real number, there are no real zeros and the graph does not cross the x-axis.

Final Answer: No real solutions

Example With One Zero

Example: x² - 6x + 9 = 0

Step 1: Plug the values into the quadratic formula

x =
-(-6) ± √-6² - 4(1)(9)
2(1)

Step 2: Simplify

x =
6 ± √0
2

x =
6 ± 0
2(1)

Step 3: Solve both

x = (6 + 0) / 2 = 3

x = (6 - 0) / 2 = 3

Both answers are the same, so there is only one zero, meaning the function only touches the x-axis once.

Final Answer: x = 3

Finding the Vertex

The vertex is the highest or lowest point of the parabola.

Vertex Formula: x = -b / 2a

How to find the coordinate point of a vertex

  • Step 1: using the given equation, substitute the b and a values into the vertex formula in order to solve for x

  • Step 2: plug the answer for x back into the original equation in order to solve for y-axis

  • Step 3: write x and y as an ordered pair → (x, y)
The vertex is at (-1.5, 2.25)

Graphing a Quadratic Function

Putting it all together, here are the steps to graph a quadratic function from an equation

  1. Find the vertex using (x = -b / 2a) and plug x into the equation to find y
  2. Find the zeros by factoring or using the quadratic formula.
  3. Plot the vertex and zeros on the graph.
  4. Draw a smooth parabola through the points.

Factoring to Solve Quadratic Equations

Sometimes quadratic equations can be solved by factoring.

Example: x² + 5x + 6 = 0

Step 1: factor the quadratic by identifying two numbers that multiply to 5 and add up to 6.

In this problem, those two numbers are 2 and 3

Step 2: split the middle term (5x) into those two numbers we identified

x² + 3x + 2x + 6 = 0

Step 3: group like terms together and factor out their GCF

(x² + 3x) + (2x + 6) = 0
x(x + 3) + 2(x + 3)

Note: you have completed this step correctly if both terms inside the parenthesis are the same.

Step 4: group the outside terms together and multiply it with the inside

(x + 2)(x + 3) = 0

Step 5: In each parenthesis, identify your zeros by solving for 0

x + 2 = 0
x = -2

x + 3 = 0
x = -3

Final Answer: x = -2 and x = -3

This means the graph crosses the x-axis at x = -2 and x = -3.

Note: not every quadratic equation is factorable. If it is not factorable, use the quadratic formula.

Systems of Linear and Quadratic Equations

A system of linear and quadratic equations consists of one or more linear and quadratic equations.

Due to the shape of a parabola, there can be more than one solution to the system.

The main method is substitution because both equations are usually equal to y.

Example:

y = x + 2

y = x² - 2x - 2

Step 1: Set the equations equal to each other.

x + 2 = x² - 2x - 2

Step 2: Move everything to one side.

0 = x² - 3x - 4

Step 3: Factor the quadratic.

Find two numbers that multiply to -4 and add to -3.

Those numbers are -4 and 1.

Split the middle term:

x² - 4x + x - 4

Group the terms:

(x² - 4x) + (x - 4)

Factor each group:

x(x - 4) + 1(x - 4)

Factor out the matching parentheses:

(x + 1)(x - 4)

Step 4: Set each factor equal to zero.

x + 1 = 0

x = -1

And:

x - 4 = 0

x = 4

Step 5: Plug each x-value into the linear equation.

Use:

y = x + 2

For x = 4:

y = 4 + 2

y = 6

So one solution is:

(4, 6)

For x = -1:

y = -1 + 2

y = 1

So the other solution is:

(-1, 1)

Final Answer: (4, 6) and (-1, 1)

These are the two points where the line and parabola intersect.

Systems of Equations

A system of equations is two or more equations that use the same variables.

Example:
y = 2x + 1
y = -x + 7

When solving a system of equations, you are trying to find the values of x and y that make both equations true at the same time.

On a graph, the solution is the point where the two lines intersect.

Solution: (x, y)

Solving Systems by Graphing

Steps

  1. Graph the first equation.
  2. Graph the second equation.
  3. Find the point where the lines intersect.
  4. Write the answer as an ordered pair: (x, y).
Example:
y = 2x + 1
y = -x + 7

These lines intersect at:
(2, 5)

Final answer: (2, 5)

Solving Systems Using Substitution

Substitution means replacing one variable with an expression from another equation.

This turns two equations into one equation with one variable.

Substitution is usually easiest when one equation already has x or y by itself.

Example 1: Basic Substitution

Solve:
y = 2x + 1
y = -x + 7

Since both equations equal y, set them equal to each other:
2x + 1 = -x + 7

Add x to both sides:
3x + 1 = 7

Subtract 1 from both sides:
3x = 6

Divide by 3:
x = 2

Plug x = 2 into one equation:
y = 2x + 1
y = 2(2) + 1
y = 4 + 1
y = 5

Final answer: (2, 5)

Solving for a Variable Before Substitution

Sometimes a variable is not already alone. In that case, isolate one variable first.

Example:
6x = -y + 23
8x + 7y = -9

The first equation has -y, so it is easiest to isolate y.

Start with:
6x = -y + 23

Add y to both sides:
y + 6x = 23

Subtract 6x from both sides:
y = -6x + 23

Substitute -6x + 23 for y in the second equation:
8x + 7(-6x + 23) = -9

Distribute:
8x - 42x + 161 = -9

Combine like terms:
-34x + 161 = -9

Subtract 161 from both sides:
-34x = -170

Divide by -34:
x = 5

Plug x = 5 into y = -6x + 23:
y = -6(5) + 23
y = -30 + 23
y = -7

Final answer: (5, -7)

Using Multiplication Before Substitution

Sometimes multiplying or dividing first makes substitution easier.

Example:
1/2x = -4 - 4y
2y - 2x = 4

The first equation has 1/2x, so multiply the entire first equation by 2 to isolate x.

2(1/2x) = 2(-4 - 4y)
x = -8 - 8y

Substitute -8 - 8y for x in the second equation:
2y - 2(-8 - 8y) = 4

Distribute:
2y + 16 + 16y = 4

Combine like terms:
18y + 16 = 4

Subtract 16:
18y = -12

Divide by 18:
y = -12/18

Simplify:
y = -2/3

Plug y = -2/3 into x = -8 - 8y:
x = -8 - 8(-2/3)
x = -8 + 16/3
x = -24/3 + 16/3
x = -8/3

Final answer: (-8/3, -2/3)

Infinitely Many Solutions

A system has infinitely many solutions when both equations represent the same exact line.

Example:
y = 2x + 3
2y = 4x + 6

Simplify the second equation by dividing everything by 2:
y = 2x + 3

Now both equations are the same:
y = 2x + 3
y = 2x + 3

Since they are the same line, every point on that line works.

Final answer: Infinitely many solutions

No Solution

A system has no solution when the lines are parallel.

Parallel lines have the same slope but different y-intercepts.

Example:
2x + y = 7
4x + 2y = 20

Rewrite both equations in slope-intercept form.

First equation:
2x + y = 7
y = -2x + 7

Second equation:
4x + 2y = 20
2y = -4x + 20
y = -2x + 10

Now compare:
y = -2x + 7
y = -2x + 10

Both lines have the same slope, -2, but different y-intercepts, 7 and 10.

Final answer: No solution

Solving Systems Using Elimination

Elimination means adding or subtracting equations so that one variable cancels out.

Elimination is easiest when the equations already have opposite terms like +2y and -2y.

Example: Elimination by Adding

Solve:
3x + 2y = 16
5x - 2y = 8

Add the equations together:
3x + 2y = 16
5x - 2y = 8

The +2y and -2y cancel out:
8x = 24

Divide by 8:
x = 3

Plug x = 3 back into one original equation:
3x + 2y = 16
3(3) + 2y = 16
9 + 2y = 16

Subtract 9:
2y = 7

Divide by 2:
y = 7/2

Final answer: (3, 7/2)

Solving a System by Multiplying One Equation

Sometimes the variables do not cancel right away. You can multiply one equation so they will cancel.

Example:
2x + 3y = 13
4x - y = 5

The first equation has +3y.
The second equation has -y.

If we multiply the second equation by 3, -y becomes -3y.

Multiply the second equation by 3:
3(4x - y) = 3(5)
12x - 3y = 15

Now the system is:
2x + 3y = 13
12x - 3y = 15

Add the equations:
14x = 28

Divide by 14:
x = 2

Plug x = 2 into one original equation:
4x - y = 5
4(2) - y = 5
8 - y = 5

Subtract 8 from both sides:
-y = -3

Divide by -1:
y = 3

Final answer: (2, 3)

Applications of Linear Systems

A linear system is a system made of linear equations.

A linear equation creates a straight line when graphed.

In Algebra 1, systems of equations and linear systems are often used to mean the same thing, but there is a small difference.

  • System of equations = two or more equations solved together
  • Linear system = two or more linear equations solved together

Linear systems can be used to solve real-life problems involving two unknowns.

Examples

  • Comparing two phone plans
  • Finding when two people will have the same amount of money
  • Comparing costs from two companies
  • Finding ticket prices
  • Solving mixture or rate problems

When Should You Use Each Method?

Use Graphing When

  • The equations are easy to graph.
  • You only need an estimated answer.
  • The intersection point is clear.

Use Substitution When

  • One variable is already by itself.
  • One equation says x = or y =.
  • It is easy to isolate one variable.
Example:
y = 3x + 2
x + y = 10

Use Elimination When

  • The equations are lined up in standard form.
  • A variable already cancels out.
  • You see opposite terms.
Example:
3x + 2y = 10
5x - 2y = 14

Use Multiplication Before Elimination When

  • The variables almost cancel, but not yet.
  • You can multiply one equation to make opposite terms.
Example:
2x + 3y = 13
4x - y = 5

Multiply the second equation by 3 to create -3y.

Simple Review

  • A system of equations has two or more equations using the same variables.
  • The solution is the point where the lines intersect.
  • Graphing means finding the intersection on a graph.
  • Substitution means plugging one equation into another.
  • Elimination means canceling out one variable.
  • Sometimes you multiply an equation first to make elimination easier.
  • Infinitely many solutions means the equations are the same line.
  • No solution means the lines are parallel.
  • Linear systems are systems made of straight-line equations.
